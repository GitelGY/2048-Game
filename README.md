🎮 2048 Game Project

גרסה מותאמת אישית של המשחק הפופולרי  2048, המבוססת על טכנולוגיות Web סטנדרטיות. המשחק כולל מערכת ניהול משתמשים מקומית, טבלת שיאים ואפקטים חזותיים.
📋 תיאור הפרויקט
הפרויקט בנוי כיישום צד-לקוח מלא (Client-side) המאפשר למשתמשים לשחק במשחק ה-2048 הקלאסי, לשמור את התוצאות שלהם ולהתחרות על המקום הראשון בטבלת השיאים.
✨ תכונות עיקריות
•	מערכת כניסה: אפשרות להזנת שם משתמש וסיסמה ושמירת הנתונים ב-LocalStorage.
•	לוח משחק אינטראקטיבי: לוח $4 \times 4$ המגיב למקשי החצים במקלדת.
•	טבלת שיאים (Leaderboard): מציגה את דירוג המשתמשים לפי הניקוד הגבוה ביותר שלהם.
•	עיצוב רספונסיבי: התאמה מלאה למכשירים ניידים ולמסכים בגדלים שונים.
•	אפקטים קוליים וויזואליים: אנימציות בעת מיזוג אריחים ועדכון הניקוד, כולל אפשרות להשתקת השמע.
•	הדפסה: כפתור ייעודי המאפשר להדפיס את מצב המשחק הנוכחי.
🛠
טכנולוגיות בשימוש

•	HTML5: מבנה דפי המשחק, דף הבית וטבלת השיאים.
•	CSS3: עיצוב המשחק, שימוש ב-Animations (כגון score-bounce ו-fadeIn) ו-Media Queries לרספונסיביות.
•	JavaScript (Vanilla): לוגיקה של המשחק, ניהול המטריצה, אלגוריתם המיזוג וניהול ה-LocalStorage.
•	Font Awesome: אייקונים ויזואליים לממשק המשתמש.
📂
מבנה הקבצים
•	html/ - דפי ה-HTML (דף הבית, דף המשחק וטבלת השיאים).
•	css/ - קבצי העיצוב (game.css, leaderboard.css, p.css).
•	js/ - הלוגיקה של המשחק וניהול המשתמשים.
•	images/ & audio/ - משאבי מדיה.
🚀 הוראות הפעלה

1.	הורד את תיקיית הפרויקט למחשב שלך.
2.	פתח את הקובץ הראשי (בדרך כלל index.html או דף הבית) בדפדפן.
3.	הזן שם משתמש וסיסמה כדי להתחיל משחק חדש.
4.	השתמש בחצי המקלדת כדי להזיז את האריחים:
5.	
o	כאשר שני אריחים עם אותו מספר נוגעים זה בזה, הם מתמזגים לאריח אחד עם סכום הערכים שלהם.
o	המטרה היא להגיע לאריח שמספרו 2048.
